Here is my tutorial to use VScode:
1. Open Vscode
2. depending if you already have a project folder create a folder you want to store your project in
3. When you are in the welcome page click open folder
4. Find and click the folder you created
5. Under the open editor file there should be the name of your folder
6. To create a file hover over your folder
7. click the paper icon that has a plus button that is the create new file button
8. When you click it, you can name the file name it whatever you want then at the end add .language you want the file to be (ex. .php, .html, .css)
9. Now you are ready to use VScode

Here is my basic github tutorial
1. Install **GIT**
2. Create a **Github** account
3. Create a **Repository**
4. **Clone** the **repository**
5. Connect to the **Remote**
6. Create a **Branch**
7. Make changes and **Commit**
8. **Push** to **Github**
9. **Fetch** updates
10. **Pull** updates
11. **Merge** your **branch** when finished
12. Fix any **Merge Conflict** if needed

Here is my Github glossary
1. **Branch:** A separate version of your project to work on changes
2. **Clone:** To copy a repository to your computer
3. **Commit:** A saved change with a message explaining what was done
4. **Fetch:** Check for updates from Github without applying them
5. **GIT:** A system that tracks changes in code
6. **Github:** A website that stores and shares Git projects online
7. **Merge:** Combine changes from one branch into another
8. **Merge Conflict:** When Git canâ€™t combine changes automatically
9. **Push:** Send your local changes to Github
10. **Pull:** Get the latest changes from Github
11. **Remote:** The online version of your repository
12. **Repository:** A folder that holds your project and its history
